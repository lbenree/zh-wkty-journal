# zh-wkty-journal

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能包含个人笔记、实验性设计、静态展示或临时记录等内容。仓库本身不针对任何特定网站或域名，仅作为 HTML 文件集合的托管与版本管理空间。

## 目录说明

```
├── README.md           # 本文件
├── index.html          # 可选：仓库首页或导航页面
├── pages/              # 存放 HTML 页面文件的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 页面引用的资源文件（如 CSS、JS、图片等）
    ├── style.css
    └── images/
```

- `pages/`：主要 HTML 页面存放位置，可根据需要进一步分类。
- `assets/`：各页面共享或独立使用的静态资源。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可直接通过浏览器打开，或通过 GitHub Pages 等服务部署访问。
- 页面内容可能随时间更新或调整，历史版本可通过 Git 提交记录追溯。
- 本仓库不保证页面的长期可用性或内容准确性，仅供归档与展示用途。
